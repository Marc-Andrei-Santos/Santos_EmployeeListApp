# Santos Employee List App

A simple ASP.NET Core Employee Management Web App with full CRUD functionality using Razor Pages and API controllers.

## Features

- Create, Read, Update, Delete Employees
- Modal-based UI (Bootstrap)
- Entity Framework Core + SQL Server
- Toast notifications on actions
- Responsive UI using Bootstrap 5
- Icons for actions (Edit/Delete)

## Tech Stack

- ASP.NET Core 7
- Razor Pages + Web API
- Entity Framework Core
- Microsoft SQL Server
- Bootstrap 5 + Icons

## How to Run

1. Download or clone this project to your computer.

2. Open the project folder in your code editor (e.g., Visual Studio, VS Code).

3. Open the `appsettings.json` file and update the database connection string to match your SQL Server setup.

4. Open a terminal or command prompt inside the project folder.

5. Run this command to update the database.

6. Run the Application.
